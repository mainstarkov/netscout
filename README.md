# 🌐 NetScout

Небольшая тулза на питоне, чтобы быстро пробить инфу по домену или айпишнику прямо из консоли. Показывает геолокацию (через несколько источников, чтобы было точнее), сканит популярные порты, достает HTTP заголовки и проверяет SSL сертификат.

## Фичи

- Переводит домен в IP и наоборот (через PTR).
- Ищет локацию сразу через 3 API (ip-api, ipwho.is, ipinfo.io). Если они выдают разные координаты, тулза считает среднее и показывает разброс.
- Сканит 10 самых нужных портов.
- Чекает, когда истекает SSL, кто выдал и кому.
- Выводит линк на Google Maps.
- Может плеваться JSON-ом, если захочешь прикрутить куда-то скриптом.

## Установка

Никаких pip install и virtualenv. Просто нужен свежий питон (3.10+):

```bash
git clone https://github.com/mainstarkov/netscout.git
cd netscout
```

## Как юзать

```bash
# Пробить домен
python3 netscout.py google.com

# Пробить IP
python3 netscout.py 8.8.8.8

# Пробить сразу несколько
python3 netscout.py google.com github.com 1.1.1.1

# Если нужно побыстрее (не сканить порты)
python3 netscout.py --no-ports ya.ru

# Вывод в JSON
python3 netscout.py --json cloudflare.com
```

## Зачем тут 3 источника геолокации?

Часто всякие сервисы гео-айпи врут, особенно для CDN или Anycast сетей. Скрипт опрашивает сразу три и выводит "Spread" (разброс). Если разброс пара километров — супер, точная инфа. Если несколько сотен километров — ну, тут уж не обессудьте, IP размазан по планете.
