# NetScout

CLI-инструмент для разведки доменов и IP-адресов. Резолвит домены, ищет обратные DNS-записи, определяет геолокацию серверов через 3 независимых источника и показывает ссылку на Google Maps.

## Возможности

- **Domain → IP** — резолв домена в IPv4/IPv6 адреса
- **IP → Domains** — обратный DNS-поиск (PTR-записи)
- **Геолокация** — страна, город, район, почтовый индекс, координаты
- **Мультиисточник** — данные с 3 API (ip-api, ipwho.is, ipinfo.io) + оценка точности
- **Сканирование портов** — 10 популярных портов (FTP, SSH, HTTP, MySQL и др.)
- **SSL-сертификат** — издатель, срок действия, subject
- **HTTP-заголовки** — сервер, статус, протокол
- **Proxy/VPN детект** — определяет прокси и датацентры
- **Google Maps** — кликабельная ссылка на местоположение сервера
- **JSON-вывод** — для скриптов и автоматизации

## Установка

Зависимостей нет — чистая стандартная библиотека Python 3.10+.

```bash
git clone https://github.com/mainstarkov/netscout.git
cd netscout
```

## Использование

```bash
# Полный скан домена
python3 netscout.py google.com

# Поиск по IP
python3 netscout.py 8.8.8.8

# Несколько целей
python3 netscout.py google.com github.com 1.1.1.1

# Без сканирования портов (быстрее)
python3 netscout.py --no-ports ya.ru

# JSON-вывод
python3 netscout.py --json cloudflare.com
```

## Как это работает

1. **DNS** — `socket.getaddrinfo()` для A/AAAA записей, `gethostbyaddr()` для PTR
2. **Геолокация** — параллельный опрос 3 бесплатных API, усреднение координат, расчёт разброса в км
3. **Порты** — TCP connect scan на 10 стандартных портов
4. **SSL** — проверка сертификата через `ssl.wrap_socket()`
5. **HTTP** — HEAD-запрос для получения серверных заголовков

## Точность геолокации

Геолокация по IP определяет город/район, а не точный адрес. Spread показывает разброс между источниками:
- 🟢 **< 50 км** — высокая точность
- 🟡 **50–200 км** — средняя
- 🔴 **> 200 км** — низкая (anycast-сети, CDN)
